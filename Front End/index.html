This app runs on the Ethereum blockchain network and holds a registry of domain names with a purchase option as Solidity smart contract.

  • The DDNS keeps information for domain names as availability, owner, associated info document, etc.
  • Users can register new available domain names which are stored on the blockchain using ETH.
  • Users can view the information for the domains.
  • Domain owners can transfer ownership and change the associated IP for already registered domains.
